快捷键

常用快捷键

alt 显示/隐藏菜单
command + shift + d / ctrl + shift + d 开发调试工具
command + shift + a / ctrl + shift + a 关于我们
command + ← / ctrl + ← 后退
command + → / ctrl + → 前进
command + r / ctrl + r / F5 刷新
command + h / ctrl + h 主页
command + t / ctrl + t 打开新标签页
command + w / ctrl + w 关闭当前标签页
command + shift + ← / ctrl + shift + ← 切换到上一个标签页
command + shift + → / ctrl + shift + → 切换到下一个标签页
command + 0 / ctrl + 0 切换到最后一个标签页
command + [1 - 9] / ctrl + [1 - 9] 切换到 指定 标签页
